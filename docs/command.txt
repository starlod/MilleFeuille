# MySQL

```
# データベースの作成
> CREATE DATABASE millefeuille CHARACTER SET utf8;
# ユーザーの追加と権限付与
> grant all privileges on millefeuille.* to millefeuille@localhost identified by 'millefeuille';
```

# Bower

```
# パッケージの最新版をインストール
$ bower install
# パッケージをまとめてアップデート
$ bower update
# インストール & bower.jsonに依存関係(dependencies)を記述
$ bower install パッケージ名 --save
# アンインストール & bower.jsonのリストから削除
$ bower uninstall パッケージ名 --save
```

`--save`オプションを付けないとbower.jsonに反映されないので注意。

# Compass

```
# ビルドしたスタイルシートの削除
$ compass clean
# ビルド
$ compass compile
# 圧縮ビルド
$ compass compile -e production
```
